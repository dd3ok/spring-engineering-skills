# Official Sources

Checked on: 2026-07-12. Spring documentation is normative for framework behavior; OWASP and NIST publications provide normative or advisory control frameworks as identified by their publishers.

- [Spring Security authorization](https://docs.spring.io/spring-security/reference/servlet/authorization/authorize-http-requests.html) — request matcher and authorization semantics.
- [Current Spring Security changes](https://docs.spring.io/spring-security/reference/whats-new.html) — current-line additions and behavior changes; switch to versioned documentation for deployed older lines.
- [Spring Security Java configuration](https://docs.spring.io/spring-security/reference/servlet/configuration/java.html) — multiple filter chains, ordering, and unmatched requests.
- [Spring Security method security](https://docs.spring.io/spring-security/reference/servlet/authorization/method-security.html) — method-level authorization and proxy/interceptor behavior.
- [Spring Security CSRF](https://docs.spring.io/spring-security/reference/servlet/exploits/csrf.html) — default protection and conditions for ignoring or disabling it.
- [Spring Security response headers](https://docs.spring.io/spring-security/reference/features/exploits/headers.html) — secure defaults and optional policies.
- [Spring Security CORS](https://docs.spring.io/spring-security/reference/servlet/integrations/cors.html) — filter ordering and preflight handling.
- [Spring Security JWT resource servers](https://docs.spring.io/spring-security/reference/servlet/oauth2/resource-server/jwt.html) — issuer, audience, keys, and authority mapping.
- [Spring Security resource-server multitenancy](https://docs.spring.io/spring-security/reference/servlet/oauth2/resource-server/multitenancy.html) — trusted issuer resolution and tenant handling.
- [Spring Security authorization server](https://docs.spring.io/spring-security/reference/servlet/oauth2/authorization-server/index.html) — current Spring Security 7+ authorization-server behavior.
- [Reactive request authorization](https://docs.spring.io/spring-security/reference/reactive/authorization/authorize-http-requests.html) and [reactive CSRF](https://docs.spring.io/spring-security/reference/reactive/exploits/csrf.html) — WebFlux security-chain behavior.
- [WebSocket security](https://docs.spring.io/spring-security/reference/servlet/integrations/websocket.html) — message authorization and CSRF considerations.
- [Spring Boot endpoint security](https://docs.spring.io/spring-boot/reference/actuator/endpoints.html) — exposure, authorization, and sanitization.
- [OWASP Application Security Verification Standard](https://owasp.org/www-project-application-security-verification-standard/) — verifiable application-security requirements.
- [OWASP Threat Modeling Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Threat_Modeling_Cheat_Sheet.html) — threat-model workflow and documentation.
- [OWASP SSRF Prevention Cheat Sheet](https://cheatsheetseries.owasp.org/cheatsheets/Server_Side_Request_Forgery_Prevention_Cheat_Sheet.html) — destination allowlists and DNS/IP validation caveats.
- [NIST Secure Software Development Framework](https://csrc.nist.gov/pubs/sp/800/218/final) — secure development practices and evidence.
