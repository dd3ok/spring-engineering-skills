# Changelog

All notable changes to Spring Engineering Skills are documented here.

## 0.1.0 - 2026-07-12

- Split the original broad skill into seven independently routable Spring engineering workflows.
- Added deterministic evidence and upgrade-plan schemas, validators, security boundaries, and routing contracts.
- Added lightweight CI, offline validation, source policy checks, and blind routing evaluation support.
- Renamed the broad static review skill to `spring-engineering-review`; retained Spring and Spring Boot best-practice terms in discovery metadata.
- Added representative Maven and Gradle multi-project golden repositories, including Spring Boot 4 with Kotlin DSL and Kotlin source.
